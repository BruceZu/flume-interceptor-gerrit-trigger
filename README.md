flume-interceptor-gerrit-trigger

Synopsis
-------------------

This is a flume interceptor for Gerrit Trigger.

Environments
-------------------

* `linux`
  * `java-1.6`
    * `gradle`

Build
-------------------

TO build library with gradle.

    ./gradlew build

|port              | 29418
|frontUrl          | **http://localhost/**
|version           | **2.8**
